# Star Wars Search Film

A site to search for movies about star wars

### Requirements
    - NodeJs 12+

### Install dependencies

```
    npm install
```
### Run project

```
    npm run start
```

### External Libraries
    - axios
    - bootstrap
    - reactstrap
    - ramda
    - react-icons
    - react-toastify
    - styled-components
